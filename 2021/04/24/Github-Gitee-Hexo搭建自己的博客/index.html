Github/Gitee+Hexo搭建自己的博客

Github/Gitee+Hexo搭建自己的博客

hexo环境配置

安装
Node.js官网
Git官网
这里要说明第一个坑：Node.js版本不能太高！
否则，你的public文件夹大小会为0，部署到github以后页面打开也会是一片空白，像这样：


我下载的是Node.js12.22.1

安装程序全部选择“next”，安装完毕后可以打开cmd测试一下 

node -v 
npm -v


Git直接下载官网右下角的安装包就行了
Git的安装程序里选择一下默认的代码编辑器，我这里选择了vscode
最后下载cnpm
国外下载比较慢，这里选择镜像 

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo

cmd输入代码 

npm install hexo-cli -g

创建博客目录，你可以选择在桌面上新建文件夹，然后在该文件夹下打开cmd/powershell输入代码 

hexo init 博客文件夹名


打开博客文件夹，在当前目录打开powershell

在powershell里输入cmd，然后输入代码hexo new 博文名
打开source-_posts，你会发现里面多出了一个文件：博文名.md
用编辑器打开，随便输入一些内容。然后在powershell中输入hexo三连： 

hexo cl
hexo g
hexo s


复制链接，在浏览器打开

按下Ctrl+C停止运行
这样，我们的hexo博客就在本地创建完成了
你可以选择在官网上下载一些好看的主题，比如我的博客主题就是樱花主题

Git的配置

注册github/gitee账号，打开cmd。
设置用户名称 

git config --global user.name "用户名"

设置用户邮箱 

git config --global user.email "用户邮箱"

生成ssh密钥 

ssh-keygen -t rsa -C "用户邮箱"

执行代码后，git会让你设置密码，可以直接回车。生成密钥后用记事本打开C盘的用户-用户名-.ssh里的id_rsa.pub，复制，然后打开github/gitee的账号设置，选择ssh密钥，密钥名随便取（如blog），然后将密钥复制进去，保存。

博客 _config.yml 文件的配置

打开博客目录的_config.yml文件，在末端加入如下代码
https：

deploy:
  type: git
  repo: https://仓库地址.git
  branch: master

ssh：

deploy:
  type: git
  repo: git@仓库地址.git
  branch: master

这两种方法都行
然后，在powershell/cmd（博客目录下）中输入代码： 

hexo d

这样，hexo博客就会生成静态文件推送到仓库

Github pages/Gitee pages配置

Github pages：
新建仓库，名字为用户名.github.io，其它的随便选。


Gitee pages：
新建仓库，名字为用户名，其它的随便选。


github pages每次推送后无需去手动更新，gitee pages推送后要自己去手动更新。
